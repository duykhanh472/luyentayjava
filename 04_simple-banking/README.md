# Simple Banking

Create Account class with balance. Methods: deposit, withdraw, show balance. Use `ArrayList<Account>` to manage multiple accounts.

## Workflow

(Kiểu tạo giả lập một hệ thống, để đăng nhập As Admin hoặc As User)

Hệ thống quản lý: `BankingManagement`

- `addAccount(String username, int password)`
- `removeAccount(Account account)`

Cho từng tài khoản: `Account`

- `deposit(int amount)`
- `withdraw(int amount)`
- `showBalance()`
